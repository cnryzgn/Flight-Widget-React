## Setup
```bash
# Before start make sure to have NODE and GIT

# Clone the project
$ git clone https://github.com/cnryzgn/Flight-Widget-React.git

# Open dir
$ cd Flight-Widget-React

# Install and Run
$ npm install && npm start

```
